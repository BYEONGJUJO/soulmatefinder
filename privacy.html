소울메이트 찾기 서비스 개인정보 보호 정책

📅 최종 업데이트: 2025년 1월 1일 | 시행일: 2025년 1월 1일

🎯제1조 (개인정보처리방침의 목적)

소울메이트 찾기(이하 "서비스")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수하고 있습니다.

🛡️ 개인정보 보호 원칙
수집 최소화, 목적 외 사용 금지, 안전한 관리, 투명한 공개를 원칙으로 합니다.

📊제2조 (개인정보의 수집 및 이용목적)

1. 수집하는 개인정보

수집항목 수집방법 수집목적 보유기간
이미지 파일 직접 업로드 AI 분석 서비스 제공 즉시 삭제
성별 선택 입력 맞춤형 분석 제공 세션 종료 시
연령대 선택 입력 연령별 분석 제공 세션 종료 시
거주지역 선택 입력 지역별 서비스 제공 세션 종료 시
IP 주소, 쿠키 자동 수집 서비스 제공, 보안 30일

2. 개인정보 이용목적

  • 서비스 제공: AI 기반 소울메이트 매칭 분석
  • 서비스 개선: 사용자 경험 향상 및 기능 개선
  • 보안 및 부정행위 방지: 서비스 오남용 방지
  • 통계 분석: 익명화된 서비스 이용 통계
⚠️ 중요 안내
업로드된 이미지는 AI 분석 처리 후 즉시 삭제되며, 어떠한 형태로도 저장되지 않습니다.

🔄제3조 (개인정보의 처리 및 보유기간)

1. 처리기간

  • 이미지 파일: 분석 완료 즉시 삭제 (최대 10초 이내)
  • 입력 정보: 브라우저 세션 종료 시 자동 삭제
  • 서비스 이용 기록: 30일 후 자동 삭제
  • 쿠키 정보: 브라우저 설정에 따라 관리

2. 보유기간 연장 사유

  • 법령에 따른 보존 의무가 있는 경우
  • 수사기관의 수사 진행 중인 경우
  • 이용자의 명시적 동의가 있는 경우
🔐 보안 강화 조치
모든 데이터는 HTTPS 암호화 통신으로 전송되며, 서버 접근은 제한된 관리자만 가능합니다.

🚫제4조 (개인정보의 제3자 제공)

서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

1. 제공 가능한 경우

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우
  • 통계작성, 학술연구 또는 시장조사를 위하여 필요한 경우로서 특정 개인을 알아볼 수 없는 형태로 가공하여 제공하는 경우

2. 제공하지 않는 정보

  • 업로드된 이미지 파일
  • 개인 식별 가능한 모든 정보
  • 분석 결과 상세 내용

🏢제5조 (개인정보처리 위탁)

서비스는 원활한 개인정보 업무처리를 위하여 다음과 같이 개인정보 처리업무를 위탁하고 있습니다:

위탁받는 자 위탁업무 내용 위탁기간
클라우드 서비스 제공업체 서버 호스팅, 데이터 저장 서비스 운영 기간
웹 분석 서비스 서비스 이용 통계 분석 서비스 운영 기간
📋 위탁업체 관리
위탁계약 시 개인정보 보호법에 따라 개인정보가 안전하게 관리될 수 있도록 필요한 사항을 규정하고 있습니다.

👤제6조 (정보주체의 권리·의무 및 행사방법)

1. 정보주체의 권리

  • 열람권: 개인정보 처리현황에 대한 열람 요구
  • 정정·삭제권: 개인정보의 정정·삭제 요구
  • 처리정지권: 개인정보 처리 정지 요구
  • 손해배상청구권: 개인정보 침해로 인한 정신적 피해 배상

2. 권리 행사 방법

  • 서면, 전화, 전자우편을 통한 연락
  • 개인정보보호법 시행령 제41조에 따른 서면양식 사용
  • 대리인을 통한 권리 행사 시 위임장 필요
⏰ 처리기간
정보주체의 권리행사 요구 시 지체없이 처리하며, 늦어도 10일 이내에 결과를 통지해드립니다.

🍪제7조 (쿠키의 설치·운영 및 거부)

1. 쿠키의 사용목적

  • 이용자 설정 정보 저장
  • 서비스 이용 통계 분석
  • 맞춤형 서비스 제공
  • 보안 강화

2. 쿠키 설정 거부 방법

  • Chrome: 설정 > 개인정보 및 보안 > 쿠키 및 기타 사이트 데이터
  • Firefox: 설정 > 개인정보 및 보안 > 쿠키 및 사이트 데이터
  • Safari: 환경설정 > 개인정보 > 쿠키 및 웹사이트 데이터
  • Edge: 설정 > 쿠키 및 사이트 권한
ℹ️ 쿠키 거부 시 영향
쿠키 설정을 거부하실 경우 일부 서비스 이용에 어려움이 있을 수 있습니다.

🔐제8조 (개인정보의 안전성 확보조치)

1. 기술적 보호조치

  • 암호화: 모든 데이터 전송 시 HTTPS 암호화
  • 접근제한: 관리자 계정 다단계 인증
  • 보안프로그램: 악성코드 탐지 및 차단
  • 로그 관리: 접근 기록 모니터링

2. 관리적 보호조치

  • 개인정보취급자 최소화
  • 정기적인 보안교육 실시
  • 개인정보 처리시스템 접근 권한 관리
  • 개인정보 처리현황 점검

3. 물리적 보호조치

  • 전산실, 자료보관실 등의 접근통제
  • 개인정보가 포함된 서류, 보조저장매체 등의 잠금장치
🛡️ 24시간 보안 모니터링
서버 보안 상태를 24시간 모니터링하여 개인정보 유출을 사전에 차단합니다.
